---
title: "GitHub Slash Commands"
date: 2025-10-18
tags: [github, productivity]
---

GitHub issues/PRs have slash commands (public preview). Type `/` in any comment to get helpers like:
- `/table` (build a table)
- `/details` (collapsible block)
- `/code` (code block)
- `/saved-replies` (supports `%cursor%`)
- `/template` (insert issue/PR templates)
