## Content Types

### Blog Posts

Traditional long-form articles with:
- Technical tutorials and guides
- Code examples and explanations
- Best practices and lessons learned
- Technology reviews and comparisons

**Format**: Markdown with syntax highlighting
**Location**: `/_posts/YYYY-MM-DD-title.md`
**Permalink**: `/blog/YYYY/MM/DD/title/`

### TIL (Today I Learned)

Short, focused posts documenting daily learnings, including:
- Code snippets and examples
- Quick tips and tricks
- Screenshots and visual aids
- Video demonstrations
- Problem-solving discoveries

**Format**: Markdown with embedded media
**Location**: `/_til/`
**Permalink**: `/til/title/`

### Projects

Portfolio items showcasing:
- Technical stack used
- Problem solved
- Implementation details
- Links to live demos and source code
- Screenshots and demonstrations

**Format**: Markdown with project metadata
**Location**: `/_projects/`
**Permalink**: `/projects/title/`

## Navigation Tips
