# Gingerly - Emergency SOS App

An iOS app designed to help users alert emergency contacts during dangerous situations with a single tap. Available on iPhone, iPad, Mac, and Apple Vision.

## Overview

Gingerly provides quick access to emergency SOS functionality through multiple methods: the app interface, Siri commands, Spotlight search, and home screen widgets. The app is designed to be fast, reliable, and accessible when seconds matter.

## Key Features

- **Multiple Access Methods**:
  - Direct app interface
  - Siri voice commands ("Hey Siri, SOS")
  - Spotlight search integration
  - Home screen widgets for one-tap access

- **Cross-Platform Support**:
  - iPhone and iPad (iOS 16.0+)
  - Mac (macOS 13.0+, Apple Silicon)
  - Apple Vision (visionOS 1.0+)

- **Multilingual Support**:
  - English, Hindi, and Telugu

- **Location Sharing**: Sends location to emergency contacts

- **Privacy-First**: No data collection

## Technical Highlights

### Platform Integration
- **Siri Shortcuts**: Voice-activated emergency alerts
- **WidgetKit**: Quick access from home screen
- **Spotlight Integration**: Launch via iOS search
- **Universal App**: Single codebase for iPhone, iPad, Mac, and Vision Pro

### Performance
- App size: Only 7 MB
- Optimized for quick launch and minimal battery impact
- Location services designed for efficiency

The app was built to provide a potentially life-saving tool that's accessible when users need it most. The multiple access methods ensure that users can trigger an alert regardless of how they typically use their device.
