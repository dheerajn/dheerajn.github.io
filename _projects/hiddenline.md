# HiddenLine - Content Analysis Tool

An iOS productivity app that analyzes various types of content to reveal hidden assumptions, blind spots, and potential risks that users might overlook.

## Overview

HiddenLine processes articles, websites, YouTube videos, documents, images, text, and code to surface deeper implications and help users identify what warrants closer examination. Results are presented in a clear, structured format designed to help users grasp core ideas quickly.

## Key Features

- **Multi-Format Analysis**:
  - Web articles and links
  - YouTube videos
  - Document uploads (PDFs, text files)
  - Images and screenshots
  - Raw text input
  - Code snippets and repositories

- **Intelligent Content Processing**:
  - Identifies hidden assumptions
  - Reveals potential blind spots
  - Highlights misleading advice
  - Surfaces potential pitfalls
  - Uncovers deeper implications

- **Clear Presentation**:
  - Structured, easy-to-understand results
  - Core ideas highlighted
  - Risk areas clearly marked
  - Actionable insights

- **Cross-Platform Support**:
  - iPhone (iOS 26.0+)
  - iPad compatibility
  - Mac (Apple Silicon)
  - Apple Vision Pro
